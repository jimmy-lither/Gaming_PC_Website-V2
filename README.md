# Gaming PC Website

A modern website for showcasing and building custom gaming PCs.

## Features

- Custom PC builder interface
- Component compatibility checking
- Price estimation
- Gallery of pre-built configurations
- Responsive design for all devices

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/jimmy-lither/Gaming_PC_Website.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Gaming_PC_Website
   ```

3. Open `index.html` in your browser to view the site.

## Development

This project uses the `dev` branch for active development.

```bash
git checkout dev
```
